CNN Pest Detection System

This deep learning-based pest classification system uses a Convolutional Neural Network (CNN) model built in Keras. It helps detect and classify agricultural pests from images with a Tkinter-based GUI for user interaction. When the model encounters unknown or untrained pest images, it reports the result as "Unknown" based on confidence thresholding.

🌟 Key Features:
  • Pest classification using trained CNN (pest.h5)
  • Tkinter-based graphical user interface (GUI)
  • Confidence thresholding to reject unknown images
  • Simple image upload and display with bounding box annotation
  • Detects common agricultural pests: Aphids, Armyworm, Beetle, Bollworm, Grasshopper, Mites, Mosquito, Sawfly, Stem Borer
  • GUI features: Image upload, Model testing, Grayscale conversion, Object detection with bounding boxes
  • Displays causes and prevention suggestions for each detected pest
  • Rejects random/unrelated images as Unknown when confidence is low
📦 Technologies & Requirements:

Python, TensorFlow, Keras, NumPy, OpenCV, Pillow, Matplotlib, scikit-learn, tkvideo, MediaPipe, gTTS, Pandas, Flask, MLxtend
